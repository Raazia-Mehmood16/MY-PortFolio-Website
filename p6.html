Project 06

Student Management System

A console-based Student Management System developed in C++ that enables administrators, teachers, and students to manage academic records through a role-based access system.

About this Project

Student Management System is a console-based application developed in C++ as a semester project for the Programming Fundamentals course at FAST-NUCES. The system provides an organized way to manage student academic records through role-based access for administrators, teachers, and students.

The application allows users to add and remove student records, assign marks, calculate CGPAs, generate performance reports, and analyze academic progress. Data is managed using file handling techniques to ensure persistence and accessibility.

The system includes features such as role-based login, student record management, marks assignment, CGPA calculation, student performance analysis, sorting functionality, and average CGPA computation. Different user roles are provided with specific functionalities to ensure structured and secure access to academic data.

Technically, the project was developed entirely in C++ using a modular programming approach. File handling was implemented for permanent data storage, while sorting and searching algorithms were used to organize and retrieve student records efficiently.

Through this project, I strengthened my understanding of C++ programming fundamentals, file handling, data processing, problem-solving, and software design principles. It also provided practical experience in building a real-world academic management application from scratch.

Key Features

  • Role-based login system for Admin, Teacher, and Student.
  • Add and delete student records.
  • Assign and update student marks.
  • Calculate and display CGPA.
  • List passed and failed students.
  • Sort students by CGPA in ascending order.
  • Sort students by CGPA in descending order.
  • Calculate average CGPA of all students.
  • Store and manage data using file handling.
  • User-friendly menu-driven console interface.
